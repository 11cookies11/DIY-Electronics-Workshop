# README Visual Assets

Use this folder for GitHub README visual materials.

## Suggested files

- `screenshots/hero-overview.png` - main homepage screenshot
- `demo/workflow-demo.gif` - short end-to-end interaction demo
- `diagrams/` - optional exported architecture images

## Notes

- Keep image size optimized for GitHub loading speed.
- Prefer `png/webp` for screenshots and short optimized `gif` for motion demo.
- If these files are missing, README image blocks will show as broken placeholders.
